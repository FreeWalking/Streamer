Введение

Программа Streamer предназначенная для просмотра видео, а также стримов формата YouTube Live, Twitch,Dailymotion, Livestream, UStream и др.(полный переченьможно изучить во вкладке "Используемые библиотеки"). Данная программа написана на Python v.3 с приминением технологии Qt. Основная библиотека для воспроизведения видео - VLC
Для более подробного изучения функций и возможностей программы перейдите в раздел "Основное"
