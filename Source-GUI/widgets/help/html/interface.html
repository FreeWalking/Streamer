Интерфейс

Интерфейс программы разделен на 3 основных области:

  • Меню
  • Основное окно
  • Строка состояния
Область "Меню" подробно рассмотрена далее в спавке.
Основное окно состоит из нескольких ключевых элементов:
  • Окно отображение видеофайла
  • Элемента старта, паузы, остановки, регулировки громкости
  • Кнопка регулирование качества видео. Становится достпуна только при просмотре стрима.
  • Кнопка открытия чата и информации о канале. Становится доступна только при просмотре стрима, а также введеных данных о пользователе и токене.
