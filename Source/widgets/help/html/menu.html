Меню

Вкладка меню содержит следующие ключевые элементы:

  • Файл
  • Настройки
  • Справка

Меню файл включает в себя
  • Открыть-Файл

    • Предназначен для открытия файлов с компьютера. Список подерживаемых форматов:

    • MPEG (ES,PS,TS,PVA,MP3)
    • AVI
    • ASF / WMV / WMA
    • MP4 / MOV / 3GP
    • OGG / OGM / Annodex
    • Matroska (MKV)
    • и др. Полный список форматов можно посмотретьпо ссылке
    >

  • Открыть-Стрим

    • Предназначен для открытия различных стримов

    • YouTube Live
    • Twitch
    • Dailymotion
    • Livestream
    • UStream
    • и др. Полный список доступен по ссылке
    Адрес стрима необходимо вводить в формате http://адрес стрима или https://адрес стрима
